Install Ruby

The install process for ruby is different based on the type of computer you are using. Install information is provided here for Windows & Mac OS. If you are using Linux, it is likely that you already have git and if not can use the package manager to easily install it.

Windows Users

If you are a Windows user you will not have ruby available by default. You will need to download and install ruby from the web. You can get it from Ruby Installer, but will need to carefully choose your version. It is recommended to use the latest 32 bit Ruby 2.2.X installer.

Make sure that you check the options to add ruby to your path.

Mac Users

If you are using a Mac, open up a Terminal window and type ruby --version. You may find that ruby is already installed, or if not that it will prompt you to install the XCode Command Line Developer Tools. If it does not, you can install ruby using RVM which will also allow you to have different versions installed. This is handy even if it is installed by default as you may want a newer version.
